Terms

ellipse

Overview

This function simply draws an ellipse on the page.

Parameters

  • Parameter 1: This parameter controls the horizontal position of the center of the ellipse.
  • Parameter 2: This parameter controls the vertical position of the center of the ellipse.
  • Parameter 3: This parameter controls the width of the ellipse.
  • Parameter 4: This parameter controls the height of the ellipse.

Notes

  • None

Examples

ellipse(203, 203, 288, 309);

rect

Overview

This function simply draws a rectangle on the page.

Parameters

  • Parameter 1: This parameter controls the horizontal position of the left side of the rectangle.
  • Parameter 2: This parameter controls the vertical position of the top of the rectangle.
  • Parameter 3: This parameter controls the width of the rectangle.
  • Parameter 4: This parameter controls the height of the rectangle.

Notes

  • None

Examples

rect(100, 100, 200, 167);

line

Overview

This function simply draws a line on the page.

Parameters

  • Parameter 1: This controls the horizontal position of one endpoint of the line.
  • Parameter 2: This controls the vertical position of one endpoint of the line.
  • Parameter 3: This controls the horizontal position of the other endpoint of the line.
  • Parameter 4: This controls the horizontal position of the other endpoint of the line.

Notes

  • None

Examples

line(0, 0, 200, 300);

text

Overview

This function writes text on the page.

Parameters

  • Parameter 1: This controls what text gets written.
  • Parameter 2: This controls the horizontal position of the left side of the text.
  • Parameter 3: This controls the vertical position of the top of the text.

Notes

  • Text written in parameter 1 should be in quotes, unless the text is a variable.

Examples

text("Hello!", 100, 200);

fill

Overview

This function controls the fill color of everything coded below it.

Parameters

  • Parameter 1: This controls how red the fill color is.
  • Parameter 2: This controls how green the fill color is.
  • Parameter 3: This controls how blue the fill color is.

Notes

  • JS code will only respond to the fill function that is closest to it as well as above it.
  • All parameter values for this function must be between 0 and 255, inclusive.

Examples

fill(178, 0, 255);

stroke

Overview

This function controls the outline color of everything coded below it.

Parameters

  • Parameter 1: This controls how red the stroke color is.
  • Parameter 2: This controls how green the stroke color is.
  • Parameter 3: This controls how blue the stoke color is.

Notes

  • JS code will only respond to the stroke function that is closest to it as well as above it.

Examples

stroke(1, 120, 120);
